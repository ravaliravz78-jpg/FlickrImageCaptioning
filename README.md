# Flickr Image Captioning

This project is a Deep Learning based Image Captioning system using CNN + LSTM architecture trained on the Flickr8k dataset.

## 🔹 Project Overview
The model generates captions for input images by:
- Extracting image features using CNN
- Generating text using LSTM

## 🔹 Files Included
- main.py → Runs the captioning system
- app.py → Web interface (if applicable)
- requirements.txt → Required Python libraries

## 🔹 Model Files (Not Included in GitHub)
The following files are required but not uploaded due to size:
- caption_model.h5
- features.pkl
- tokenizer.pkl
- max_length.pkl
- dataset/

## 🔹 Installation

Install dependencies:

pip install -r requirements.txt

## 🔹 Run the Project

python main.py
